# Dinajpur Image

A responsive static website showcasing the festivals, culture, and photography of Dinajpur, Bangladesh.

## About

Dinajpur Image is a media brand dedicated to capturing and sharing the rich cultural heritage, vibrant festivals, and stunning photography of Dinajpur. This website serves as a digital gallery and information hub for the region.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **5 Pages**: Home, Festivals, Culture, Gallery, and Contact
- **Hero Banner**: Eye-catching hero section with gradient background
- **Image Cards**: Beautiful card layouts showcasing festivals and cultural elements
- **Photo Gallery**: Masonry-style grid layout for displaying images
- **Contact Form**: Functional contact form with validation
- **Smooth Animations**: Scroll animations and hover effects
- **Mobile Menu**: Hamburger menu for mobile navigation

## Pages

1. **Home**: Welcome page with hero banner, about section, and featured cards
2. **Festivals**: Showcases major festivals celebrated in Dinajpur
3. **Culture**: Highlights cultural traditions, music, dance, and heritage
4. **Gallery**: Photo gallery with responsive grid layout
5. **Contact**: Contact form and information

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- Vanilla JavaScript (ES6+)
- No frameworks or dependencies

## Deployment

This website is ready for GitHub Pages deployment. To deploy:

1. Go to repository Settings
2. Navigate to Pages section
3. Select the branch (e.g., `main` or `copilot/create-static-website-dinajpur-image`)
4. Set root directory to `/` (root)
5. Save and wait for deployment

The site will be available at: `https://[username].github.io/Dinajpur-Image/`

## Local Development

To run locally, simply open `index.html` in a web browser, or use a local server:

```bash
# Using Python 3
python3 -m http.server 8080

# Using Node.js
npx http-server -p 8080
```

Then navigate to `http://localhost:8080`
